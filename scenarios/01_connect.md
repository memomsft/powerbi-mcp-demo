# Escenario 01 - Conectar al Semantic Model

## Prompt

```
Connect to semantic model 'SalesModel' in Fabric Workspace 'powerbi-mcp-demo'
```

## Qué hace

El MCP autentica con tus credenciales de Microsoft Entra ID y establece conexión con el modelo. Puede abrir el navegador para login la primera vez.

## Resultado esperado

```
Successfully connected to the semantic model 'SalesModel' 
in the Fabric workspace 'powerbi-mcp-demo'.
```

## Tool ejecutada

`connection_operations`
