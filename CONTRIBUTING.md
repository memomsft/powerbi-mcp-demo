# Contribuciones

Este repositorio documenta un ejercicio exploratorio en evolución. Si replicas el demo y encuentras hallazgos diferentes, errores o mejoras, puedes contribuir de las siguientes formas:

## Reportar hallazgos

Abre un **Issue** en GitHub con:
- Escenario donde ocurrió
- Modelo AI usado (GPT-4.1, Haiku, Sonnet, otro)
- Prompt utilizado
- Resultado obtenido vs resultado esperado

## Proponer mejoras

1. Haz fork del repositorio
2. Crea una rama: `git checkout -b mejora/descripcion`
3. Realiza tus cambios
4. Abre un **Pull Request** describiendo qué cambió y por qué

## Áreas donde se agradecen contribuciones

- Resultados con modelos no probados (GPT-5, GPT-4o, etc)
- Escenarios adicionales (jerarquías, perspectivas, calculation groups)
- Experiencia con Power BI Desktop o archivos PBIP en lugar de Fabric
- Hallazgos con versiones más recientes del MCP (actualmente v0.4.0)
